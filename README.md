# FitNation - Your Personal Fitness Companion

A comprehensive fitness tracking application to help you achieve your health and wellness goals.

## Features

- **Workout Tracking** - Log and track your daily workouts
- **Diet Management** - Monitor your nutrition and meal plans
- **Profile Management** - Track your fitness journey and progress
- **Personalized Recommendations** - Get AI-powered diet suggestions

## Run Locally

**Prerequisites:** Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
